# sc-sketches

Each sketch is a work in progress — not a finished piece, but a record of sound being made.

Part of **igram lab | 音響音楽研究所**.

## Running

Each `.scd` file is self-contained. Open it in SuperCollider (or your editor of choice), then evaluate the blocks in order — boot and SynthDefs first, then the performance block.

Some sketches use UGens from [sc3-plugins](https://github.com/supercollider/sc3-plugins) (e.g. JPverb, Greyhole). Install them if a sketch doesn't run.

Tuning note: sketches may use non-standard tunings (Sketch 01 is tuned to A=430Hz).
